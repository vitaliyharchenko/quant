Программа курса

Введение числа 14 занятий

Все темы ⇲

Тема 1. Подбор мерок, удобных для измерения данной величины, и подбор величин, удобных для измерения данной меркой.

Тема 2. Задача опосредованного сравнения: с помощью посредника, равного одной из величин, с помощью посредника-меры. Число как результат измерения.

Тема 3. Выбор меры, удобной для измерения длины, площади, объёма, массы, углов, количества.

Тема 4. Ребусы, последовательности, разрезание. Танграм. Задачи, решаемые методом «обратный ход»

Тема 5. Подбор мерок, удобных для измерения данной величины, и подбор величин, удобных для измерения данной меркой. Простые и составные мерки. Подбор подходящих предметов, используемых в качестве мерки

Тема 6. Число как результат измерения величины и как средство для ее восстановления. Компоненты действия измерения: величина (А), мерка (Е), число (Л) и связь между ними.

Тема 7. Запись числа как результата измерения и счета с помощью меток, считалок и с помощью цифр в различных нумерациях. Построение величины по мерке и числу; подбор и изготовление мерки по заданной величине и числу. Зависимость одного из трех компонентов (~ = Л) от изменения другого при постоянном Е третьем (фактически речь идет о функциональной зависимости).

Сложение и вычитание чисел 6 занятий

Все темы ⇲

Тема 8. Решение и составление математических выражений, уравнений и задач с заменой буквенных данных на числовые данные (в пределах десятка) и наоборот. Сказочные цифры

Тема 9. Задачи на измерение (отмеривание). Повторение способа измерения величины с помощью мерки. Ситуация, когда величина намного больше мерки. Набор мерок для измерения такой величины. Выбор подходящей мерки

Тема 10. Числовая и буквенная закономерность.. Числовая и буквенная закономерность. Логические задачи для юных математиков

Многозначные числа 10 занятий

Все темы ⇲

Тема 11. Измерение величины с помощью набора мерок. Запись результата измерения. Задачи обратные данной (нахождение мерки по величине и количеству, нахождение количества по величине и мерке)

Тема 12. Позиционные системы счисления. Понятие многозначного позиционного числа как результата измерения величины системой мерок с заданным отношением (основание системы). Чтение и запись чисел в различных системах счисления. Место нуля в записи многозначных чисел

Тема 13. Десятичная система счисления как частный случай позиционной системы счисления. Чтение и запись любых многозначных чисел. Названия первых четырех разрядов. Сравнение многозначных чисел. Действия с многозначными числами

Тема 14. Умножение как способ измерения величин, связанный с переходом в процессе измерения к новым меркам. Постановка и решение задач, приводящих к изменению единиц измерения. Графическое изображение умножения. Оценка различных отношений между величинами и исходной меркой

Тема 15. Составные уравнения. Составление уравнений. Решение нестандартных задач
